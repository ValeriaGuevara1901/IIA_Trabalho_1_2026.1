📚 Sobre o Projeto

Este sistema implementa um modelo de recomendação por conteúdo utilizando o algoritmo TF-IDF (Term Frequency - Inverse Document Frequency) para recomendar livros da literatura brasileira.

🔬 Como Funciona

  • Cada livro possui 3 características: autor, tema e categoria
  • O algoritmo TF-IDF calcula a relevância das palavras em cada perfil
  • A similaridade de cosseno mede a semelhança entre preferências
  • Recomendações são baseadas nos livros melhor avaliados pelo usuário

📊 Dados do Sistema

  • 50 livros da literatura brasileira
  • 100 usuários de treinamento
  • Matriz de utilidade com 5.000 avaliações
  • Escala de avaliação: 1 a 5 estrelas
